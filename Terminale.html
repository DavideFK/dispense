Terminale unix (Bash)

Comandi

ls

Visualizza l'elenco dei file nella cartella corrente.
ls -la Visualizza anche i file nascosti.

Equivalente di dir su DOS.

touch

touch "filename" Crea un file di nome "filename".

mkdir

mkdir "percorso" Crea una cartella di nome "percorso" dove specificato nel percorso stesso.

mv

mv "filename" "percorso/filename" sposta il file "filename" dalla directory corrente al percorso/file "percorso/filename".

rm

rm "filename" Elimina il file "filename" dalla directory corrente.


ls -r "percorso" Elimina la cartella "percorso" dalla directory corrente.

code

code "filename/percorso" Avvia Visual Code da Terminale.
Con "filename/percorso" puoi specificare un file/percorso da aprire immediatamente all'avvio dell'editor.
