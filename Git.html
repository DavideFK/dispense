PROCEDURE

Aggiornare il sitema

  1. sudo apt update
  2. sudo apt upgrade

Installare Repository sudo add-apt-repository ppa:git-core/ppa

Installare Git e controllare la versione

  1. sudo add-apt-repository ppa:git-core/ppa
  2. sudo apt update
  3. sudo apt install git

configurare user name

configurare user email

  1. git config --global user.name "YOUR USERNAME"
  2. git config --global user.email "YOUR EMAIL"

Configurare il nuovo branch principale come main

Migliorare l'interfaccia con il colore

Controllare i parametri di login

  1. git config --global init.defaultBranch main
  2. git config --global color.ui auto
